1. Crear Firestore

Ve a la consola de Firebase, ingresa a Firestore Database y haz clic en Crear Base de Datos.

⚠️ Selecciona "Modo de Prueba" para habilitar lecturas y escrituras temporalmente.
2

2. Configurar Config

Copia las credenciales Web App de Firebase e introdúcelas en el archivo local:

js/firebase-config.js
3

3. Subir Productos

Cuando el archivo config tenga tus credenciales válidas, presiona el botón de abajo para migrar todos tus productos.
